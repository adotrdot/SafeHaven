## SafeHaven
==============================

SafeHaven is a Qt5 application that replicates a diary.


## Requirements
==============================

- qt >= 5.11.3 (to install and configure, follow the guide here https://zetcode.com/gui/qt5/introduction/ide)
- libqt5multimedia5
- libqt5multimedia5-plugins
- fontconfig

## Building
==============================

1. Clone this repo
1. Run `qmake`
1. Run `make`

## Running
==============================

Run `./safehaven`
